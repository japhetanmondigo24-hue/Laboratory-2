Day 1: Coding Experience

Worked on improving my PHP login system with secure password handling. Today's session was productive—I implemented password hashing using bcrypt and added input validation to prevent SQL injection attacks. The key takeaway was understanding how crucial security is when handling user credentials.

Day 2: Late Night Study

It's an exhausting day for me as I grind to finish my quiz reviewer for our course tomorrow. I wish myself luck! Despite the fatigue, I managed to review database normalization concepts and practiced writing efficient queries. The late-night sessions always seem to help things click into place.

Day 3: UI/UX Design Challenge

Today I focused on frontend development, specifically improving the user interface for a personal project. I experimented with CSS Grid and Flexbox to create responsive layouts. The challenge was balancing aesthetic design with functionality—not everything that looks good on desktop works on mobile!

Day 4: GitHub Collaboration

Collaborated with team members on GitHub using branches and pull requests. I learned the importance of clear commit messages and proper documentation. Code reviews from my peers helped me identify bugs I would have missed otherwise. Teamwork really does make the dream work!
